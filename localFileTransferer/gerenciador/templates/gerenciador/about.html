Sobre o Local File Transferer

Nossa Missão

O Local File Transferer foi criado com a intenção de fazer com que as transferências de arquivos entre dispositivos sejam facilitadas e agilizadas.

O funcionamento consiste na conectividade entre os dispositivos de forma local.

Nossa aplicação permite que você:

  • Faça upload de arquivos facilmente
  • Compartilhe arquivos na rede local
  • Gerencie seus arquivos de forma segura
  • Transfira dados rapidamente

Recursos

  • Conectividade de forma local
  • Download de pastas em .zip
  • Upload de arquivos entre dispositivos
  • Criação de perfis
  • Página de administração
  • Visualização de logs das transferências
  • Segurança local
